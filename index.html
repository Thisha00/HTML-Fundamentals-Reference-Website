What is the Internet?

The Internet is a worldwide network of interconnected computer networks that allows devices to communicate and exchange information using TCP/IP.

Key Characteristics

  • Global network
  • Uses TCP/IP protocols
  • Supports communication worldwide
  • Provides access to websites and online services
  • Connects billions of devices

How the Internet Works

Client

A client is a user's device or application that requests information from a server.

Server

A server stores websites, applications, and data and responds to client requests.

Client–Server Model

  1. User enters a website address.
  2. Browser contacts DNS.
  3. DNS returns the server IP address.
  4. Browser sends an HTTP/HTTPS request.
  5. Server processes the request.
  6. Server sends HTML files.
  7. Browser displays the webpage.

HTTP

HTTP (HyperText Transfer Protocol) transfers web pages between browsers and servers.

HTTPS

HTTPS is the secure version of HTTP that encrypts data using SSL/TLS.

Domain Name System (DNS)

DNS translates human-readable domain names into IP addresses so browsers can locate web servers.

When you type a website address, your computer sends a request to a DNS server to find the corresponding IP address. The DNS server looks up the address and returns the correct IP address to your computer. Your browser then uses that IP address to connect to the website's server and load the webpage.

Why DNS is Important?

  • Easy to use
  • Enables internet communication
  • Improve performance
  • Supports scalability
  • Provides redundancy and reliability
  • Helps with security
